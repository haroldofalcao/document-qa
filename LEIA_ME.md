# Documentação do Projeto AppSheet - Medical Visits

Esta pasta contém todo o histórico técnico e guias de implementação do projeto de App de Visitas Médicas.

## Conteúdo

*   **Guia_Inicial.md**: Instruções passo-a-passo para a configuração inicial do AppSheet e da Planilha Google.
*   **Guia_Ciclo_2.md**: Instruções para a segunda fase de melhorias (restrição de cadastro e datas de alta).
*   **Script_Email.gs**: Código do Google Apps Script utilizado para o envio automático de e-mails.
*   **Estrutura_Banco_Dados.md**: Definição das colunas e tabelas do banco de dados.
*   **Resumo_Projeto.md**: Visão geral do projeto e instruções de "handover" (passagem de bastão).
*   **Plano_Implementacao_Ciclo_2.md**: O plano técnico aprovado para o Ciclo 2.

## Como usar

Se você estiver trabalhando com um assistente de IA no futuro, você pode pedir para ele:
"Leia os arquivos na pasta `Documentacao_AppSheet` para entender o contexto do projeto."
